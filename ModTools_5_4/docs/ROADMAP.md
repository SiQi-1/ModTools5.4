# ModTools 5.4 路线图（2026-03 更新）

> 本文档用于同步“代码真实状态”和“下一步优先级”。
> 详细历史请看 `CHANGELOG.md`。

## 一、当前已完成（Done）

### 1) 工程与工作区框架
- `.CIV` 工程结构、序列化与加载流程已落地。
- 主窗口已接入新建/打开/保存工程。
- 工作区树导航已支持直接节点与子条目组切换。

### 2) 分类编辑能力
- 已接入子条目编辑器：
  - 文明、领袖、区域、建筑、单位、改良设施、总督、伟人
  - 政策卡、项目、信仰
- 相关分类已接入 SQL 预览（多数同时支持 XML 预览转换）。

### 3) 生成与输出
- 工程根可生成 Data/Text/美术相关输出文件。
- 文本工作区已统一承载 Text.sql / Text.xml 预览。

## 二、当前缺口（TODO）

### P1：议程（Agendas）完整接入
- 现状：议程仍走占位/未接入输出。
- 目标：补齐“编辑器 -> 预览 -> 生成输出”全链路。

### P1：全局搜索接入真实逻辑
- 现状：搜索页“全局搜索”只有分页壳。
- 目标：按分类提供可检索结果，并支持定位到对应工作区条目。

### P2：导入能力补齐
- 现状：仅部分分类有导入。
- 目标：补齐文明、领袖、政策卡、项目、信仰、议程导入流程。

### P2：测试与回归
- 现状：自动化测试覆盖较少。
- 目标：新增最小冒烟测试（启动、工程读写、关键预览生成）。
